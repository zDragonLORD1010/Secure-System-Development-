# –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –∏ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã (`+`, `-`, `/`, '*', `=`, `//`, `%`)

–ï—Å–ª–∏ –ø—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏, —Ç–æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ —ç—Ç–æ —è—â–∏–∫–∏, –≤ –∫–æ—Ç–æ—Ä—ã–µ –≤—ã –º–æ–∂–µ—Ç–µ –ø–æ–ª–æ–∂–∏—Ç—å –ª—é–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. **–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –º–æ–∂–Ω–æ –Ω–∞–∑–≤—ã–≤–∞—Ç—å –∫–∞–∫ —É–≥–æ–¥–Ω–æ**, –≥–ª–∞–≤–Ω–æ–µ, —á—Ç–æ–±—ã –º–µ–∂–¥—É –Ω–∞–∑–≤–∞–Ω–∏–µ–º –Ω–µ –±—ã–ª–æ –ø—Ä–æ–±–µ–ª–æ–≤ (–≤–º–µ—Å—Ç–æ –ø—Ä–æ–±–µ–ª–∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `_`, –Ω–∞–ø—Ä–∏–º–µ—Ä `string_123`) –∏ —Å–∏–º–≤–æ–ª–æ–≤ –ø–æ —Ç–∏–ø—É ('+', '-', '@', –∏ —Ç.–¥).

## –ü—Ä–∏—Å–≤–æ–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è (—á–µ–º—É —Ä–∞–≤–Ω–∞ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –∏–ª–∏, –ø—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏, `=`)

## –°–ª–æ–∂–µ–Ω–∏–µ (`+`)

–°–∫–ª–∞–¥—ã–≤–∞–µ—Ç —á–∏—Å–ª–∞, —Å–∏–º–≤–æ–ª—ã, —Å—Ç—Ä–æ–∫–∏. –ì–ª–∞–≤–Ω–æ–µ - –æ–±–æ–∑–Ω–∞—á–∏—Ç—å —á—Ç–æ –≤—ã –¥–µ–ª–∞–µ—Ç–µ: —Å–∫–ª–∞–¥—ã–≤–∞–µ—Ç–µ —Å—Ç—Ä–æ–∫–∏, —á–∏—Å–ª–∞, —Å–∏–º–≤–æ–ª—ã –∏ —Ç.–¥.

–ù–∞–ø—Ä–∏–º–µ—Ä, –ø—É—Å—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è `a` –±—É–¥–µ—Ç —Ä–∞–≤–Ω–∞ `10`, –∞ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è b - `20`, —Ç–æ–≥–¥–∞:

```py
a = 10    # 10 - —á–∏—Å–ª–æ
b = 20    # 20 - —á–∏—Å–ª–æ
c = a + b # –Ø –∑–∞–ø–∏—Å—ã–≤–∞—é –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é —Å—É–º–º—É a –∏ b, c –±—É–¥–µ—Ç —Ä–∞–≤–Ω–æ 30 (10 + 20)
```

**–í–ê–ñ–ù–û!** –ï—Å–ª–∏ –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö `a` –∏ `b` –±—É–¥—É—Ç –ª–µ–∂–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ `10` –∏ `20`, –∞ `'10'` –∏ `'20'` —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, —Ç–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å–ª–æ–∂–µ–Ω–∏—è –±—É–¥–µ—Ç –¥—Ä—É–≥–æ–π:

```py
a = '10'  # '10' - —Å—Ç—Ä–æ–∫–∞
b = '20'  # '20' - —Å—Ç—Ä–æ–∫–∞
c = a + b # –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –≤ c –∑–∞–ø–∏—à–µ—Ç—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∏—Ö —Å–∫–ª–µ–∏–≤–∞–Ω–∏—è, —Ç–æ –µ—Å—Ç—å '10' + '20' = '1020' –∏ c –±—É–¥–µ—Ç —Ç–∞–∫–∂–µ —Å—Ç—Ä–æ–∫–æ–π
```

## –í—ã—á–∏—Ç–∞–Ω–∏–µ (`-`), –¥–µ–ª–µ–Ω–∏–µ (`/`), —É–º–Ω–æ–∂–µ–Ω–∏–µ (`*`)

–≠—Ç–∏ —Ç—Ä–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ –º–æ–∂–Ω–æ –ø—Ä–æ–≤–æ–¥–∏—Ç—å —Ç–æ–ª—å–∫–æ —Å –¥–≤—É–º—è —á–∏—Å–ª–∞–º–∏ (`integer` –∏–ª–∏ `float`), –Ω–∞–ø—Ä–∏–º–µ—Ä:

```py
a = 100     # 100 - —á–∏—Å–ª–æ
b = 2       # 2 - —á–∏—Å–ª–æ

c_1 = a - b # –í —Å_1 —Ä–∞–∑–Ω–∏—Ü–∞ a –Ω–∞ b, —Ç–æ –µ—Å—Ç—å c_1 –±—É–¥–µ—Ç —Ä–∞–≤–Ω–æ 98 (100 - 2)

c_3 = a * b # –í —Å_3 –∑–∞–ø–∏—à–µ—Ç—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç —É–º–Ω–æ–∂–µ–Ω–∏—è a –Ω–∞ b, —Ç–æ –µ—Å—Ç—å c_3 –±—É–¥–µ—Ç —Ä–∞–≤–Ω–æ 200 (100 * 2)
```

**–ù–û!** –°—Ç—Ä–æ–∫—É –º–æ–∂–Ω–æ —É–º–Ω–æ–∂–∏—Ç—å –Ω–∞ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ, –∏ —Ç–æ–≥–¥–∞ –æ–Ω–∞ –ø–æ–≤—Ç–æ—Ä–∏—Ç—å—Å—è —Å—Ç–æ–ª—å–∫–æ —Ä–∞–∑, —Å–∫–æ–ª—å–∫–æ –≤—ã —Å–∫–∞–∂–∏—Ç–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä:

```py
a = "Hello" # "Hello" - —Å—Ç—Ä–æ–∫–∞
b = 3       # 3 - —á–∏—Å–ª–æ

c = a * b   # –í c –∑–∞–ø–∏—à–µ—Ç—Å—è —Å—Ç—Ä–æ–∫–∞ a b —Ä–∞–∑, —Ç–æ –µ—Å—Ç—å —Å –±—É–¥–µ—Ç —Ä–∞–≤–Ω–æ "HelloHelloHello" ("Hello" * 3)
```
 
## `mod` (`%`) –∏ `div` (`//`)

`mod` - —ç—Ç–æ –æ—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è –æ–¥–Ω–æ–≥–æ —á–∏—Å–ª–∞ –Ω–∞ –¥—Ä—É–≥–æ–µ

`div` - —Ü–µ–ª–∞—è —á–∞—Å—Ç—å –æ—Ç –¥–µ–ª–µ–Ω–∏—è –æ–¥–Ω–æ–≥–æ —á–∏—Å–ª–∞ –Ω–∞ –¥—Ä—É–≥–æ–µ

–ù–∞–ø—Ä–∏–º–µ—Ä —É –º–µ–Ω—è –≤ –∫–æ—Ä–æ–±–∫–µ `box` –ª–µ–∂–∏—Ç 10 —è–±–ª–æ–∫ (üçéüçéüçéüçéüçéüçéüçéüçéüçéüçé) —è —Ö–æ—á—É –∏—Ö —Ä–∞–∑–¥–µ–ª–∏—Ç—å –º–µ–∂–¥—É —Ç—Ä–µ–º—è –ª—é–¥—å–º–∏: `Ivan`, `Masha`, `Sasha`. –¢–æ–≥–¥–∞, –∫–∞–∂–æ–º—É —è –º–æ–≥—É –¥–∞—Ç—å –ø–æ 3 —è–±–ª–æ–∫–∞: `Ivan` (üçéüçéüçé), `Masha` (üçéüçéüçé), `Sasha` (üçéüçéüçé), –∞ –æ–¥–Ω–æ —è–±–ª–æ–∫–æ (üçé) —É –º–µ–Ω—è –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –≤ –∫–æ—Ä–∑–∏–Ω–µ, –ø–æ—Ç–æ–º—É —á—Ç–æ —è —Ö–æ—á—É —Ä–∞–∑–¥–µ–ª–∏—Ç—å –º–µ–∂–¥—É –Ω–∏–º–∏ —è–±–ª–æ–∫–∏ –ø–æ—Ä–æ–≤–Ω—É. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –µ—Å–ª–∏ –º—ã —Ä–∞–∑–¥–µ–ª–∏–º 10 –Ω–∞ 3 –º—ã –ø–æ–ª—É—á–∏–º: `10 / 3 = 3 (1 –æ—Å—Ç–∞–ª–æ—Å—å)`, –≥–¥–µ `3` - —ç—Ç–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–≥–æ –¥–µ–ª–µ–Ω–∏—è `div`, –∞ `1` - —Ä–µ–∑—É–ª—å—Ç–∞—Ç –ø–æ–¥—Å—á–µ—Ç–∞ –æ—Å—Ç–∞—Ç–∫–∞ `mod` –æ—Ç 10 —è–±–ª–æ–∫ –º–µ–∂–¥—É 3 –ª—é–¥—å–º–∏. –í–æ—Ç –∫–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ `Python`:

```py
box = 10     # 10 - —á–∏—Å–ª–æ, —Å–∫–æ–ª—å–∫–æ —è–±–ª–æ–∫ –ª–µ–∂–∏—Ç –≤ –∫–æ—Ä–∑–∏–Ω–µ
b = box // 3 # –í b —è –ø–æ—Å—á–∏—Ç–∞–ª —Å–∫–æ–ª—å–∫–æ —è–±–ª–æ–∫ —è –º–æ–≥—É –¥–∞—Ç—å –∫–∞–∂–¥–æ–º—É, —á—Ç–æ–±—ã –±—ã–ª–æ –ø–æ—Ä–æ–≤–Ω—É, —Ç–æ –µ—Å—Ç—å b —Ä–∞–≤–Ω–æ 3 (10 // 3)
c = box % 3  # –í —Å —è –ø–æ—Å—á–∏—Ç–∞–ª, —Å–∫–æ–ª—å–∫–æ —è–±–ª–æ–∫ –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –≤ –∫–æ—Ä–∑–∏–Ω–µ, —Ç–æ –µ—Å—Ç—å c —Ä–∞–≤–Ω–æ 1 (10 % 3)
 ```

# –í–≤–æ–¥ –∏ –≤—ã–≤–æ–¥ –ø—Ä–æ—Å—Ç—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö (`input()` –∏ `print()`)

`input()` - –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤–≤–µ—Å—Ç–∏ —á–∏—Å–ª–æ, —Å—Ç—Ä–æ–∫—É, —Å–∏–º–≤–æ–ª –∏ —Ç.–¥.

`print()` - –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–≤–æ–¥–∏—Ç —á—Ç–æ —É–≥–æ–¥–Ω–æ –≤ —Ñ–æ—Ä–º–∞—Ç–µ —Å—Ç—Ä–æ–∫–∏ (`string`)

`int()` - –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–µ—Ä–µ–≤–æ–¥–∏—Ç —Å—Ç—Ä–æ–∫—É –≤ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ (`integer`), –Ω–∞–ø—Ä–∏–º–µ—Ä, –∏–∑ —Å—Ç—Ä–æ–∫–∏ ''10'' –º—ã –ø–æ–ª—É—á–∏–º —á–∏—Å–ª–æ `10`

`float()` - - –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–µ—Ä–µ–≤–æ–¥–∏—Ç —Å—Ç—Ä–æ–∫—É –≤ –¥—Ä–æ–±–Ω–æ–µ —á–∏—Å–ª–æ (`float`), –Ω–∞–ø—Ä–∏–º–µ—Ä, –∏–∑ —Å—Ç—Ä–æ–∫–∏ `10.123` –º—ã –ø–æ–ª—É—á–∏–º —á–∏—Å–ª–æ `10.123`

`str()` - –∫–æ–º–∞–Ω–¥–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–µ—Ä–µ–≤–æ–¥–∏—Ç —á—Ç–æ —É–≥–æ–¥–Ω–æ –≤ —Å—Ç—Ä–æ–∫—É (`string`), –Ω–∞–ø—Ä–∏–º–µ—Ä –∏–∑ —á–∏—Å–ª–∞ `1000.101` –º—ã –ø–æ–ª—É—á–∏–º —Å—Ç—Ä–æ–∫—É `'100.101'`

–î–∞–≤–∞–π—Ç–µ —Ä–∞–∑–±–µ—Ä–µ–º—Å—è –∫–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ –∫–æ–¥–µ:

```py
a = input()  # –ü—É—Å—Ç—å –≤ –∞ –≤–≤–æ–¥–∏—Ç—Å—è —á–∏—Å–ª–æ `10`, —Ç–æ–≥–¥–∞ Python –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é a `'10'`
a = int(a)   # –ü–æ—ç—Ç–æ–º—É –º—ã —É—Ç–æ—á–Ω—è–µ–º, —á—Ç–æ `10` —ç—Ç–æ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ –∫–æ–º–∞–Ω–¥–æ–π int(), –∏ –≤ –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ a –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –ø—Ä–µ–¥—ã–¥—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, —Ç–æ–ª—å–∫–æ –¶–ï–õ–û–ï, —Ç–æ –µ—Å—Ç—å `a = 10`, –∞ –Ω–µ `a = '10'`

# –ò–õ–ò —Ç–æ–∂–µ —Å–∞–º–æ–µ –º–æ–∂–Ω–æ –∑–∞–ø–∏—Å–∞—Ç—å —Å –ø–æ–º–æ—â—å—é

a = int(input())

b = input()  # –ü—É—Å—Ç—å –≤ b –≤–≤–æ–¥–∏—Ç—Å—è —Å—Ç—Ä–æ–∫–∞ "Hello", —Ç–æ–≥–¥–∞ Python –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é a "Hello"

print(a)     # –í—ã–≤–æ–¥–∏—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –∞ –∏ –ø–µ—Ä–µ–Ω–æ—Å–∏—Ç—Å—è –Ω–∞ —Å–ª–µ–¥—É—é—â—É—é —Å—Ç—Ä–æ–∫—É. –¢–æ –µ—Å—Ç—å –≤ –∫–æ–Ω—Å–æ–ª–∏ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ 10

print(a, b)  # –í—ã–≤–æ–¥–∏—Ç –∑–Ω–∞—á–µ–Ω–∏—è –∞ –∏ b —á–µ—Ä–µ–∑ –ø—Ä–æ–±–µ–ª –∏ –ø–µ—Ä–µ–Ω–æ—Å–∏—Ç—Å—è –Ω–∞ —Å–ª–µ–¥—É—é—â—É—é —Å—Ç—Ä–æ–∫—É. –¢–æ –µ—Å—Ç—å –≤ –∫–æ–Ω—Å–æ–ª–∏ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ 10 Hello
```

**–î–∞–≤–∞–π—Ç–µ —Ä–µ—à–∏–º –∑–∞–¥–∞—á—É:** –í–≤–æ–¥–∏—Ç—Å—è 2 —Ü–µ–ª—ã—Ö —á–∏—Å–ª–∞ a –∏ b, –Ω—É–∂–Ω–æ –ø–æ—Å—á–∏—Ç–∞—Ç—å –∏—Ö —Å—É–º–º—É –∏ –≤—ã–≤–µ—Å—Ç–∏ `"–°—É–º–º–∞ = {—Å—É–º–º–∞_a+b}. –£—Ä–∞!"`:

```py
# –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ –º—ã –≤–≤–µ–ª–∏ 3 –∏ 2 —Å —Ä–∞–∑–Ω—ã—Ö —Å—Ç—Ä–æ–∫ –≤ –∫–æ–Ω—Å–æ–ª–∏

a = input()  # –í–≤–æ–¥–∏–º a –∏ b (–ü–æ–∫–∞ –ø—Ä–æ–≥—Ä–∞–º–º–∞ —Å—á–∏—Ç–∞–µ—Ç –∏—Ö —Å—Ç—Ä–æ–∫–∞–º–∏)
b = input()

a = int(a)   # –£—Ç–æ—á–Ω—è–µ–º, —á—Ç–æ a –∏ b —ç—Ç–æ —Ü–µ–ª—ã–µ (integer) —á–∏—Å–ª–∞
b = int(b)

summ = a + b # –ó–∞–ø–∏—Å—ã–≤–∞–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å–ª–æ–∂–µ–Ω–∏—è a –∏ b –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é summ

print("–°—É–º–º–∞ = " + str(summ) + ". –£—Ä–∞!")

# –ò–õ–ò —Ç–æ–∂–µ —Å–∞–º–æ–µ –º–æ–∂–Ω–æ –≤—ã–≤–µ—Å—Ç–∏ —Å –ø–æ–º–æ—â—å—é

print(f`–°—É–º–º–∞ = {summ}. –£—Ä–∞!`)
```

# –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –∏ —É—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã (`if`, `elif`, `else`)

```py

    –¥–µ–π—Å—Ç–≤–∏–µ_1
elif (—É—Å–ª–æ–≤–∏–µ_2):
    –¥–µ–π—Å—Ç–≤–∏–µ_2

    –¥–µ–π—Å—Ç–≤–∏–µ_3
```

–£ —Ü–∏–∫–ª–∞ `for` –µ—Å—Ç—å 3 –∑–Ω–∞—á–µ–Ω–∏—è:

```py
    a = a - 1 # –û—Ç–Ω–∏–º–∞–µ—Ç –æ—Ç a –µ–¥–∏–Ω–∏—Ü—É
```

–í —ç—Ç–æ–π –∑–∞–¥–∞—á–µ –º—ã –≤–≤–µ–ª–∏ –¥–≤–∞ —Ü–µ–ª—ã—Ö —á–∏—Å–ª–∞ (`a = 10` –∏–∑ –ø–µ—Ä–≤–æ–π —Å—Ç—Ä–æ–∫–∏ –≤–≤–æ–¥–∞, —á–∏—Å–ª–æ `b = 6` –∏–∑ –≤—Ç–æ—Ä–æ–π —Å—Ç—Ä–æ–∫–∏ –≤–≤–æ–¥–∞). –î–∞–ª—å—à–µ –Ω–∞—á–∏–Ω–∞–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å —Ü–∏–∫–ª:

# –°–∏–º–≤–æ–ª—ã –∏ —Å—Ç—Ä–æ–∫–∏

–†–∞–∑–±–µ—Ä–µ–º—Å—è –∫ –∫–∞–∫–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É/—ç–ª–µ–º–µ–Ω—Ç–∞–º —Å—Ç—Ä–æ–∫–∏ –º—ã –æ–±—Ä–∞—â–∞–µ–º—Å—è `string` - —Å—Ç—Ä–æ–∫–∞:

- `string[a]` - –°–∏–º–≤–æ–ª —Å—Ç—Ä–æ–∫–∏ –ø–æ–¥ –Ω–æ–º–µ—Ä–æ–º `a`. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ —Å—Ç—Ä–æ–∫–∞ `string = "Hello"`, –∞ `a = 1`, —Ç–æ `string[a] = 'e'` (–ø–æ—Ç–æ–º—É —á—Ç–æ —É —Å–ª–æ–≤–∞ `"Hello"` `'H'` –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å `0`, `'e'` –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å `1`, `'l'` –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å `2`, `'l'` –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å `3`, `'o'` –∏–º–µ–µ—Ç –∏–Ω–¥–µ–∫—Å `4`)

- `string[0:-a]` -  –ù–µ—Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤ –∫—Ä–æ–º–µ –ø–æ—Å–ª–µ–¥–Ω–∏—Ö `a` —Å–∏–º–≤–æ–ª–æ–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ —Å—Ç—Ä–æ–∫–∞ `string = "Hello"`, –∞ `a = 2`, —Ç–æ `string[0:-a] = "Hel"`

# –û–¥–Ω–æ–º–µ—Ä–Ω—ã–µ —Å–ø–∏—Å–∫–∏ (–º–∞—Å—Å–∏–≤—ã) –≤ Python

## –í–≤–æ–¥ –∏ –≤—ã–≤–æ–¥ –æ–¥–Ω–æ–º–µ—Ä–Ω—ã—Ö —Å–ø–∏—Å–∫–æ–≤ (–º–∞—Å—Å–∏–≤–æ–≤)

# –î–≤—É–º–µ—Ä–Ω—ã–µ —Å–ø–∏—Å–∫–∏ (–º–∞—Å—Å–∏–≤—ã / –º–∞—Ç—Ä–∏—Ü—ã) –≤ Python

```py
M = int(input()) # –í–≤–æ–¥–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫ –≤ –º–∞—Ç—Ä–∏—Ü–µ

for i in range(N):
    row = [] # –°—Ç—Ä–æ–∫–∞ –≤ –º–∞—Ç—Ä–∏—Ü–µ (–û–¥–Ω–æ–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤)
    for j in range(M):
        row.append(0)
    matrix.append(row) # –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏ –≤ –º–∞—Ç—Ä–∏—Ü—É

# –°–æ–∑–¥–∞–Ω–∏–µ –¥–≤—É–º–µ—Ä–Ω–æ–≥–æ –º–∞—Å—Å–∏–≤–∞ (–º–∞—Ç—Ä–∏—Ü—ã) —Ä–∞–∑–º–µ—Ä–∞ N –Ω–∞ M, –ø—Ä–∏ –≤–≤–æ–¥–µ –∫–∞–∂–¥–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏
N = int(input()) # –í–≤–æ–¥–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç–æ–ª–±—Ü–æ–≤ –≤ –º–∞—Ç—Ä–∏—Ü–µ
M = int(input()) # –í–≤–æ–¥–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫ –≤ –º–∞—Ç—Ä–∏—Ü–µ
matrix = [] # –ú–∞—Ç—Ä–∏—Ü–∞ (–î–≤—É–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤)
for i in range(N):
    row = [] # –°—Ç—Ä–æ–∫–∞ –≤ –º–∞—Ç—Ä–∏—Ü–µ (–û–¥–Ω–æ–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤)
    for j in range(M):
        row.append(input()) # –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ —Å—Ç—Ä–æ–∫—É
    matrix.append(row) # –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏ –≤ –º–∞—Ç—Ä–∏—Ü—É


# –°–æ–∑–¥–∞–Ω–∏–µ –¥–≤—É–º–µ—Ä–Ω–æ–≥–æ –º–∞—Å—Å–∏–≤–∞ (–º–∞—Ç—Ä–∏—Ü—ã) —Ä–∞–∑–º–µ—Ä–∞ N –Ω–∞ len(row), –ø—Ä–∏ –≤–≤–æ–¥–µ —Å—Ç—Ä–æ–∫ —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏
N = int(input()) # –í–≤–æ–¥–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç–æ–ª–±—Ü–æ–≤ –≤ –º–∞—Ç—Ä–∏—Ü–µ
matrix = [] # –ú–∞—Ç—Ä–∏—Ü–∞ (–î–≤—É–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤)
for i in range(N):
    row = input().split(" ") # –í–≤–æ–¥–∏–º —Å—Ç—Ä–æ–∫—É —Ä–∞–∑–¥–µ–ª–µ–Ω–Ω—É—é —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª–µ–º
    matrix.append(row)

# –í—ã–≤–æ–¥ –º–∞—Ç—Ä–∏—Ü—ã (–ö–∞–∫ —Ç–∞–±–ª–∏—Ü–∞)
for i in range(N):
    for j in range(M):
        if j == M-1:
            print(matrix[i][j])
        else:
            print(matrix[i][j], end=" ")
```

# –§—É–Ω–∫—Ü–∏–∏
